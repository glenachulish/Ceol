Ceol – User Guide

      
Ceol is your personal trad music library. Here's a quick overview of everything it can do.


      

        
Navigation

        
LibraryBrowse, search and filter all your tunes. Click any tune to open it.

        
SetsGroup tunes into named sets for a session or practice run. Play them in sequence.

        
NotesFreeform documents — ideas, practice notes, links, attachments.

        
AchievementsAuto-logged progress events (rating changes, hitlist adds) plus your own notes.

      


      

        
Adding tunes

        
ABC FileUpload a .abc file to bulk-import many tunes at once.

        
Paste ABCPaste raw ABC notation directly. Each tune must start with X:.

        
TheSession.orgSearch the world's largest trad tune database, preview sheet music, then save.

        
FlutefFling.scotBrowse the FlutefFling session catalogue and import tunes with one click.

        
DropboxConnect a Dropbox folder and import .abc files or link audio files.

      


      

        
Library features

        
Search & filtersFilter by title, type (reel, jig…), key, mode, or star rating.

        
⭐ Star ratingRate tunes 1–5 stars to track how well you know them. Achievements are logged when your rating improves.

        
📌 HitlistFlag tunes you want to learn. Use the Hitlist button to filter to just those tunes.

        
Select modeSelect 2+ tunes to group them as versions of the same tune, or delete in bulk.

      


      

        
Inside a tune

        
Sheet Music tabRenders ABC notation with abcjs. For FlutefFling tunes, shows the PDF instead.

        
Audio playbackPlay the tune as MIDI. Set a bar range to loop a section for practice.

        
Notes tabAdd free text notes. URLs are auto-linked — paste an MP3 or YouTube link to get an inline player.

        
VersionsGroup different settings of the same tune. Click a parent tune to see all its versions.

      


      

        
Library management

        
💾 BackupExport your entire library — tunes, sets, notes, and attachments — as a .zip file.

        
📂 Import LibraryRestore from a backup file. Replaces your current library, so backup first.
