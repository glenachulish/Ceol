Ceòl Mobile – Guide

      
Your personal trad music library, optimised for mobile.


      

        
Navigation

        
♪ LibraryBrowse, search and filter all your tunes. Tap any tune to open it.

        
≡ SetsGroup tunes into named sets for a session or practice run.

        
📁 CollectionsThematic groupings — a tune can belong to multiple collections.

        
📝 NotesFreeform documents — ideas, practice notes, links, attachments.

        
⭐ ProgressAuto-logged achievements plus your own notes.

        
☰ MenuImport tunes, export/import library backup, help.

      


      

        
Adding tunes

        
＋ buttonOpens the Import dialog. Choose from ABC file, TheSession.org, FlutefFling, Dropbox, or paste ABC notation.

      


      

        
Library features

        
⚙ FilterTap the gear icon to expand filters: type, key, mode, star rating.

        
⭐ Star ratingRate tunes 1–5 stars. Tap a star inside any tune to update.

        
📌 HitlistFlag tunes you want to learn. Filter to hitlist only via ⚙ Filters.

      


      

        
Inside a tune

        
Sheet MusicRenders ABC notation. Tap bars to set a practice loop range.

        
Notes tabAdd free text notes. Paste an MP3 or YouTube link for inline playback.

        
VersionsDifferent settings of the same tune, grouped together.
