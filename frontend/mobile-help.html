Finding tunes in your library

The Library tab is your main view. Tunes appear as cards with the title, type (reel, jig, etc.), key.

Searching

Tap the search bar at the top and start typing. Search looks at the tune title, alternative names, and composer. It searches as you type — no need to hit a button.

Filters

The filter row sits under the search bar. You can filter by:

  • Type — reel, jig, hornpipe, slow air, etc.
  • Key — G major, D minor, etc.
  • Mode — major, dorian, mixolydian.
  • Rating — minimum stars (1–5).
  • Hitlist — only tunes on your current hitlist.
  • Favourites — only ones you've marked as favourite.
  • Collection — only tunes in a specific collection.

Filters combine — you can have "Reels in D, rated 4 or above, on the hitlist" all at once. Tap a filter again to clear it. You can clear these options by hitting the little cog on the right-hand side — this gives you a more open view of the tunes in your library.

Reading a tune

Tap any tune card to open the full view. You'll see:

  • Sheet music — rendered from the ABC notation. Pinch to zoom.
  • Play / pause / stop controls at the top and bottom.
  • Tempo control — slow it down for learning.
  • Transpose — shift the tune up or down a number of semitones. The sheet music updates live. You can save versions.
  • Metadata — type, key, composer, where it came from, any notes you've added.

Practising — the loop, tempo, hitlist

The bar-range loop

On the sheet music, tap a bar to start the range, then tap another bar to end it.

The playback now loops just those bars over and over until you tell it to stop.

Combine that with the tempo control in order to master a tricky section of the tune.

Highlights

You can highlight specific bars on a tune for any reason.

Adding tunes from TheSession.org

There are a few ways to import from The Session. Here is one method:

  • Open TheSession.org in a browser, find the tune.
  • On the tune page, scroll to a setting you like. Each setting shows ABC notation in a code-style block starting with X:1.
  • Select the whole ABC block — from X:1 down to the last bar of the tune — and copy.
  • In Ceòl, open the menu (☰ top-left) and tap ➕ Import tunes.
  • Tap the Paste ABC tab.
  • Paste into the text box. Tap Import.

The tune lands in your library with title, type, key, mode, and composer all filled in from the ABC headers.

Other ways to add tunes

Open the menu (☰) and tap ➕ Import tunes. You'll see a row of tabs across the top. The most useful besides Paste ABC:

  • ABC File — upload one or more .abc files from your phone.
  • Folder — upload a whole folder of ABC files at once. Good for migrating from another app.
  • Photos — take a photo of sheet music.
  • PDF Files / PDF Book — pull tunes out of PDF tunebooks.
  • Ceol file — import a .ceol.json export from another Ceòl user.
  • TheCraic — import from the TheCraic iOS app.

The other tabs (Music Scanner, YouTube, FlutefFling, Dropbox) all work too — explore them as you need them.

When you import in bulk, Ceòl shows a Recently Imported set at the top of your Sets view, grouping the batch together so you can find what you just added.

Adding media to a tune

Upload an audio file

  • Open the tune.
  • Scroll down to the attachments section.
  • Tap Add audio.
  • Tap Choose audio file and pick the file from your phone's storage.
  • Wait for the upload. MP3, M4A, WAV, OGG, AAC, FLAC all work.

Once attached, the file appears with a play button and a label ("audio 1", "audio 2", etc — you can rename them). Tap to play in-line.

Link a YouTube video

Same Add audio menu, switch to the YouTube tab, paste the URL.

Add a sheet music photo

From the tune view, find the sheet music attachment button. Either choose a photo from your library or take one with the camera. Useful when a printed version has fingering, ornament marks, or notes written on it.

Sets

Ceòl has two ways to build sets.

Build a set from scratch (freeform)

  • Tap Sets at the bottom.
  • Tap + New Set.
  • Give it a name (and optional notes) and tap Create.
  • Open the new set. Search your library and add tunes one by one in the order you want to play them.
  • Drag to reorder if needed.

Build a set with Circle of Fifths help

If you're not sure what might work well together, Ceòl can suggest:

  • In the Sets view, tap Build a Set.
  • Pick a starting tune.
  • Ceòl shows you tunes in compatible keys — the same key, the relative minor/major, neighbours on the circle of fifths.
  • Add the ones you like, in order.
  • Name and save.

Useful when you've got a big library and want to find unexpected combinations.

Playing a set

Open a set and tap play — Ceòl plays through every tune in sequence. You can also listen to the transitions between each pair of tunes.

For uninterrupted offline playback (lock screen, no data, car Bluetooth) see Playlists for offline play below.

Playlists for offline play

The Playlists tab in the bottom nav is for chaining sets together into a single uninterrupted run of music — useful for driving, walking, or anywhere your phone might go to sleep mid-tune.

The trick is that each set first needs to be rendered to an MP3 in advance. That way the playlist plays as a regular audio stream — the lock screen shows controls, the screen can sleep, the data connection can drop, and the music keeps going.

Rendering a set for offline play

  • Open a set from the Sets tab.
  • Scroll down to the 📱 Offline play section.
  • Tap 🎵 Render for offline play. Your phone does the work — it can take 10–60 seconds depending on the set length. Keep the screen on while it runs.
  • When it finishes you'll see ✓ Ready to play offline and a ▶ Play full set (offline) button.

If you later change the tunes in a set, or edit the ABC of any tune in it, the offline audio is marked ⚠ Audio out of date. Tap 🎵 Re-render to refresh it.

Building a playlist

  • Tap Playlists at the bottom.
  • Tap + New Playlist, give it a name (e.g. "Driving North"), tap Create.
  • Open the new playlist. Tap + Add a set and pick from your library. Each set shows a badge:
    • ✓ Ready — rendered and ready to play.
    • 🎵 Needs rendering — no offline audio yet.
    • ⚠ Out of date — rendered audio is stale.
  • Sets without rendered audio show an inline 🎵 Render button right there in the picker — tap it and the render happens in place. The picker stays open so you can fix several sets in one go.

Sets in the playlist can be reordered (drag) and removed.

Playing a playlist

Open a playlist and tap the big play button. The player slides up from the bottom of the screen with prev/next buttons, a progress bar, and an Up next list. Sets play in order with a brief silence between them.

The iPhone lock screen shows the current set name, the artist "Ceòl", and play/pause/skip controls — so you can leave the phone in your pocket and control playback from the lock screen, AirPods, or car stereo.

Sets without offline audio are skipped automatically with a brief "Skipped: [Set Name]" note. Render them first if you want them included.

Collections

Collections are looser groupings than sets, for instance Diggers tunes.

Creating a collection

  • Tap Collections at the bottom.
  • Tap + New Collection.
  • Name it. Tap Create.

Adding tunes to a collection

Open the collection and use the Add Tunes button to search your library and pick. A tune can be in as many collections as you like — they're just labels, not folders. From the sheet music for a tune you can add it to a set or a collection.

Filtering by collection

Back in the Library, the Collection filter lets you see only tunes in a specific collection.

Versions — keeping different settings

Lots of traditional tunes have multiple settings: Ceòl lets you keep more than one as related rows so the library doesn't fill up with what look like duplicates.

What versions do

One tune is the parent; the others are versions of it. They appear stacked on a single card in the library with a count ("3 versions"). Open the card and switch between them.

How to make a version

  • When importing a tune that's clearly another setting of one you already have, Ceòl will offer to attach it as a version rather than a new tune.
  • You can also manually merge two tunes via the versions panel on a tune detail page.
  • Each version can have a version label ("my setting", "Liz Carroll's version", "Diggers version") so you remember which is which.

Achievements

The Achievements tab shows a timeline of milestones — first tune added, first set built, tune count crossing 50/100/500, that sort of thing.

The menu (☰)

The hamburger icon (☰) at the top-left of every screen opens the side menu. Things tucked away here:

  • 📝 Notes — free-form notes you've attached to tunes, sets, or kept as standalone documents.
  • ⭐ Achievements — milestones timeline.
  • ➕ Import tunes — the import screen (TheSession, ABC files, photos, PDFs, etc.). This used to live in the bottom nav; it's here now so the bottom nav stays focused on playback.
  • 📤 Export / 📂 Import — full library backup as a .json file you can save to Files or iCloud. Useful before big changes or for moving between devices.
  • 📱 Export for TheCraic — generates a file the TheCraic iOS app can read.
  • ♻️ Rationalise Library — runs a tidy-up pass on duplicate detection and version grouping.
  • Appearance — auto / light / dark theme.
  • Logout — sign out (worth doing on a shared device).

Your account & password

Forgot your password?

From the login page, tap Forgot password? under the password field. Enter your username or email.

An admin gets notified, and they'll send you a reset link out-of-band (iMessage, WhatsApp, whatever).

Tap the link, set a new password, you're in.

Logging out

There's a Logout option in the main menu. Worth doing if you're on a shared device.
